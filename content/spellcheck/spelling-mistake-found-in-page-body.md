The text contained in the `<body>` element of your document is visible to users.

# How do I fix this?

Edit the source code that generates the HTML page containing the spelling error.

```html
<html>
  <head>
    <title>My Webpage</title>
  </head>
  <body>
    <!-- <h1>My Headinf</h1> -->
    <!-- <h1>My Heading</h1> -->
  </body>
</html>
```
